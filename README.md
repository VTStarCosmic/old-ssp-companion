# old-ssp-companion
The files for my SSP companion in SSP (not JS/TS + React/Python)

Unfortunately, the version of SSP I was using to develop is so old that it's been flagged a security hazard on my PC.
This upload is for viewing purposes only, and I would not recommend downloading SSP and running it without a VM.

For those unfamiliar with SSP, the main coding content is in the ghost file.  For a normal user, there would be multiple ghosts with unique data in this folder.
Thus the name formatting.  The other folders contain code that is NOT originally written by me, just edited.

The Emily4/Hunter ghosts are not my own, and any reference to them is just leftover data required for the app to run on my VM.

Bob ghost last edited by me in 2022.
